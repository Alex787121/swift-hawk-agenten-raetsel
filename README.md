Swift & Hawk — Agentenrätsel

Dieses Paket ist vorbereitet für ein GitHub-Repository namens 'swift-hawk-agenten-raetsel' und GitHub Pages.

Dateien:
- index.html
- README.md
- firebase-config.example.js (Beispiel, um Online-Bestenliste zu nutzen)
- assets/ok.wav, assets/wrong.wav, assets/complete.wav (Platzhalter)
- assets/placeholder.png (Platzhalterbild)
- assets/qr_github_pages.png (QR zum erwarteten Pages-URL)

Upload-Anleitung:
1) Erstelle auf GitHub ein neues Repository mit dem Namen 'swift-hawk-agenten-raetsel'.
2) Entpacke diese ZIP und lade alle Dateien in das Repository hoch.
3) Aktiviere GitHub Pages: Settings → Pages → Branch: main → Folder: / (root).
4) Die Seite wird erreichbar sein unter: https://Alex787121.github.io/swift-hawk-agenten-raetsel/

Firebase (optional):
- Erstelle ein Firebase-Projekt, aktiviere Firestore und erstelle eine Web-App.
- Kopiere deine Firebase-Konfiguration in eine Datei 'firebase-config.js' (siehe firebase-config.example.js).
- Lade firebase-config.js ins Repo; aktiviere im Spiel die Checkbox "Aktiv: Online-Bestenliste".
